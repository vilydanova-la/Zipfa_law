# Zipfa_law
# Анализ текста по закону Зипфа

Этот проект реализует проверку закона Зипфа для заданного текста на языке Python.  
Программа:

- считает частоты слов в папке `text`;
- сортирует слова по убыванию частоты и присваивает им ранг;
- подбирает оптимальную константу закона Зипфа методом наименьших квадратов;
- вычисляет теоретические частоты по модели \( F(R) = \frac{C}{R} \);
- выводит результаты в виде таблицы и строит график «ранг–частота».

## Используемые технологии

Проект использует только стандартные библиотеки Python и `matplotlib`:

- `re` — выделение слов из текста с помощью регулярных выражений;
- `collections.Counter` — подсчёт частот встречаемости слов;
- `matplotlib.pyplot` — построение графиков.
